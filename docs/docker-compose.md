# Команды работы с docker-compose

Список запущенных compose-контейнеров:

    docker-compose ps

Список запущенных/остановленных compose-контейнеров:

    docker-compose ps -a
    
Запуск compose-контейнера в console:

    docker-compose up

Запуск compose-контейнера в виде daemon:

    docker-compose up -d
